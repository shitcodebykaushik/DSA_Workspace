 # C++ functions
 - The c++ standard lib contains the following functions that are based on binary search and work in the logristhmix time .
 - `Lower_bound` It returns a pointer to the first array element whose value is at least x .
 - `Upper_bound` It returns a pointer to th first array elements whose value is larger than x .
 - `Equal_range` Returns both above pointers .
 - The function assusme that the array is sorted , if there is no such element the pointer points to the element after the last array element .
